# alura-studies

Aplicação criada no curso **React: Escrevendo com Typescript** da plataforma **Alura**.

## Aprendizado

- Criação de **function components**
  
- Carregamento de **listas dinâmicas**
  
- Utilização de **atributos customizados** de componentes
  
- Utilização de **CSS Modules**
  
- Utilização do *Hook* **useState** para controle de mudanças de estado em propriedades
  
- Reaproveitamento de **interfaces**
  
- **Bind** de funções à eventos de componentes
  

## Requisitos

- [Npm](https://nodejs.org/pt-br/)
  

## Criação do projeto através do npm

No diretório do projeto, digite:

```bash
npx create-react-app . --template typescript --use-npm
```

## Execução da aplicação

```bash
npm install #para instalação de dependências do projeto
npm start #execução do servidor de aplicação
```
